# Tasks
Приложение Задачи.
Реализована возможность:
1. Создания новой записи по кнопке, чтобы пользователь мог сразу вводить текст.
2. Удаления записи по нажатию на круглую кнопку через 3 секунды
3. Сохранения данных с помощью CoreData
